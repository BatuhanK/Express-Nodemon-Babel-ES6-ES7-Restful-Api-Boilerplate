# Starting point for a new Nodejs Restful Api

### How to use ?

##### Clone the project and install dependencies

```
git clone https://github.com/hqro/Express-Nodemon-Babel-ES6-ES7-Restful-Api-Boilerplate My-App
cd My-App
npm install
```

##### Development mode

```
npm run dev
```

##### Production mode

```
npm run build
npm start
```
